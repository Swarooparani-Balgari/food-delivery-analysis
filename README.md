# Food Delivery Data Analysis

## Objective
Analyze Delivery orders to undersatnd top-selling cuisines, revenue by city, revenue by restaurant, and delivery performance.

## Tools Used
- Excel
- SQL
- Python(Pandas, Matplotlib)
- Power BI

## Dataset
The dataset contains:
- OrderID
- OrderDate
- City
- Restaurant
- Cuisine
- DeliveryTime_Min
- Orders
- AvgPrice

## Analysis Performed
- Calculated total revenue
- Analyzed revenue by restaurant
- Compared revenue across cities
- Analyzed top-selling Cuisines
- Created visualizations for business insights

## Key Insights
- Indian Cuisine dominates the other cuisines
- Mumbai had the highest revenue among all cities
- Revenue distribution varies significantly across different dates
- Spice Hub restaurant had the highest revenue among all other restaurants

## Files in Repository
- Dataset.xlsx - Dataset and charts
- Queries.sql - SQL analysis queries
- analysis.py - Python analysis and visualization
